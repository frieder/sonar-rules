Make sure not to use any nullable annotations besides those provided by the Checker Framework.

Noncompliant Code Example

import javax.annotation.Nonnull;

class MyClass {
    @Nonnull
    private Object field;

    ...
}

Compliant Solution

import org.checkerframework.checker.nullness.qual.NonNull;

class MyClass {
    @NonNull
    private Object field;

    ...
}
