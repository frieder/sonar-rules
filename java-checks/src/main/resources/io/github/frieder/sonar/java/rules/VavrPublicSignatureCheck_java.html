Do not use vavr.io.* classes in public API. Try to keep the interfaces as generic as possible. Internal use of vavr classes is still possible.

Noncompliant Code Example

interface MyIface {
    io.vavr.collection.List<String> func1();
    void func2(io.vavr.collection.Map<String,Object> param);
}

Compliant Solution

interface MyIface {
    java.util.List<String> func1();
    void func2(java.util.Map<String,Object) param);
}
