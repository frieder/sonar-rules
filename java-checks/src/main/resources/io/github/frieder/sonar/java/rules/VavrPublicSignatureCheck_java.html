Do not use vavr.io.collection.* in public API. Use regular java.util.* classes instead.

Noncompliant Code Example

interface MyIface {
    io.vavr.collection.List<String> func1();
    void func2(io.vavr.collection.Map<String,Object> param);
}

Compliant Solution

interface MyIface {
    java.util.List<String> func1();
    void func2(java.util.Map<String,Object) param);
}
